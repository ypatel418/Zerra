# Zerra

Zerra is an open source file sharing web application. Users are able to upload files, and share them with other Zerra users with ease via email address. The main reason why we created Zerra was to learn, and get more comfortable with Spring Boot web, as well as React and React Component Libraries.

Although Zerra has a small set of user features for now, we are planning on adding more such as a file preview feature and a folder system.

The current deployment of Zerra is on fly.io for the backend, and Vercel for the frontend.

[Click Here](https://zerra-five.vercel.app/dashboard) to visit Zerra!
## Features

- Ability to share files with other users
- See files only shared with you
- Change email and password via settings
- Delete files



## Tech Stack

**Client:** React, Firebase, MUI React Component Library

**Server:** Java, Spring Boot Web, MySQL, Hibernate, LOMBOK,
